## 簡介

以 Vue.js 框架實作網站，功能區分為前、後台：  
* 前台  
  *   首頁：圖片輪播、網站導覽、Q&A  
  *   關於我們：網站介紹  
  *   產品列表：種類選擇、查看產品詳情  
  *   產品詳細：選擇產品數量並加入購物車、相關產品  
  *   購物車結帳：訂單成立流程  
* 後台  
  *   管理者登入：帳號及密碼驗證  
  *   產品管理：新增、修改、刪除及上／下架產品資訊  
  *   訂單管理：查看訂單資訊
