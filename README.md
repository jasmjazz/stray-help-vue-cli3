# Stray_Help(Vue-Cli3)

### Demo

https://jasmjazz.github.io/stray-help-vue-cli3/dist/#/


### 簡介

以 Vue.js 框架實作網站，功能區分為前、後台：

...* 首頁：圖片輪播、平台介紹
...* 糧食列表：種類選擇
...* 糧食詳細：加入購物車、相關糧食


### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
