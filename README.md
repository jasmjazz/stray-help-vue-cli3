# Stray-Help(Vue-Cli3)

## Demo

https://jasmjazz.github.io/stray-help-vue-cli3/dist/#/


## 簡介

以 Vue.js 框架實作網站，功能區分為前、後台：  
* 前台  
  *   首頁：圖片輪播、網站導覽  
  *   關於：網站介紹  
  *   產品列表：種類選擇、查看產品詳情  
  *   產品詳細：選擇產品數量、加入購物車、相關商品  
  *   購物車結帳：訂單成立流程  
* 後台  
  *   管理者登入：帳號及密碼驗證
